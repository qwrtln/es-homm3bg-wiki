## Notas

- Blocked Field cannot be entered, but can be exited.
- [^1] Si a un héroe se le permite entrar en un campo bloqueado debido a un efecto, sólo podrá *pasarlo*. Esto significa que deben utilizar inmediatamente otro punto de movimiento durante el mismo turno para salir del campo bloqueado. Si no pueden salir del campo, no se les permite entrar en él.
- [^1] Si un héroe necesita retirarse a un campo bloqueado, puede hacerlo, siempre que tenga un punto de movimiento de sobra que pueda utilizar para salir del campo bloqueado durante el mismo turno. Si no le quedan puntos de movimiento, se retira al campo en el que estaba antes de moverse al campo bloqueado.
- Un héroe no puede terminar su turno en un campo bloqueado.
- [^1] Un campo ocupado por el héroe de un jugador puede ser utilizado por el otro héroe del jugador para *pasar a través*. Esto significa que el héroe necesita tener otro punto de movimiento disponible después de moverse al campo. Los campos ocupados por los héroes del jugador siguen las mismas reglas que los campos bloqueados.
