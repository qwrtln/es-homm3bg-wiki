# Inicio

## :attack: What Is This? { data-toc-label='What Is This?' }

Una sencilla colección de las cartas contenidas en Heroes of Might and Magic 3: El Juego de Mesa, que también ofrece explicaciones extra sobre cómo funcionan algunas cartas o interactúan con otras.

De vez en cuando quiero ver lo que hacen algunas cartas, ya sea mientras teorizo sobre qué jugar, o incluso durante el juego, cuando quiero comprobar o hacer referencia a algo. Esta base de datos pretende ofrecer una forma sencilla de encontrar cualquier carta del juego, incluyendo una referencia visual e información adicional cuando sea necesario o aplicable.


## :defense: Future Plans? { data-toc-label='Future Plans?' }
