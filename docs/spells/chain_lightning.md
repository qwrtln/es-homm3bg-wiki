## Notas

- El :damage: de Relámpago en Cadena también se aplica a unidades aliadas.
- Si varias unidades se encuentran a la misma distancia del objetivo de este hechizo, el lanzador elige cuál de ellas recibe el segundo daño y cuál el tercero.
