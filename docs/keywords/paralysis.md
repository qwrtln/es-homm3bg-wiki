#  :paralysis: parálisis


## Notas

- Una unidad paralizada omite su próxima activación.En lugar de activar, la: Parálisis: Token se elimina de él.
- La: Parálisis: el token se elimina prematuramente si la unidad paralizada recibe daño.Un ataque que no aplica ningún daño no cuenta para esto.
- Si la unidad paralizada recibió daños por un ataque, tomará represalias (si no se hace antes durante esta ronda de combate).
